## React JS chatbot UI
React JS + RTK Query

## Available features
- [x] Chat.
- [x] Dashboard.
- [x] Prompt management.
- [ ] Data tables check.
- [ ] Custom Knowledge base.
- [ ] Profile

## Installation
1. Install node Version >= 20
   https://nodejs.org/en/blog/release/v20.9.0
2. Git clone and run
```bash
npm install
```

## Run
**Run local:**
```bash
npm start
```

## Run with docker
**Docker compose:**
```bash 
docker-compose up -d --build
```
